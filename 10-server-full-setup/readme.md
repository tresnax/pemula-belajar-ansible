# Server Full Setup

- Web Server (apache & php)
- Database Server (Mysql/postgresql)
- FTP Server (vsftpd)
- MalDet (Malware Detected)
- ClamAV (Antivirus)
- MMonit (Monitoring System)
- Mod_Evasive (Anti DDOS)
- Firewall
